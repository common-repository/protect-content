== Description ==

This is a really simple plugin that let you chose for each post to have image protection and/or text protection

== Installation ==

1. Upload the `protect-content` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
1. That's it!! Now whenever you edit a post or create a new one, you will see a small panel on the bottom right of the screen that shows the applicable options.

== Frequently Asked Questions ==

= Can I disable/enable for all of my posts? =

Yes you can disable/enable the protection for all of your posts in option panel of the plugin which is called Text & Image Protector which is a submenu of settings.

= What does this plugin do? =

This plugin let the user protect text and/or images of any post that the user wants.
